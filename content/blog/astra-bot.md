## The Vision Behind Astra Bot

Astra Bot started as a simple idea: create a Discord bot that does everything, and does it well. Today, it serves over **30+ Discord servers**.

### Key Features

- **Moderation System** - Auto-mod, warnings, bans/mutes, logs
- **Music System** - High-quality streaming via Lavalink
- **Economy & Leveling** - Currency, XP, rewards, leaderboards
- **Utility** - Tickets, giveaways, polls, statistics

### Technical Stack

- Discord.js v14
- Node.js & MongoDB
- Redis for caching
- Lavalink for audio
- React dashboard
