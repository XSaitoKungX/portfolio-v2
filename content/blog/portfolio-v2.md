## A New Era of Portfolio Development

**Portfolio v2** built with the latest web technologies.

### Technical Stack

- TanStack Start 1.149
- React 19.2 & TypeScript 5.9
- Bun runtime
- Tailwind CSS v4 & shadcn/ui
- Appwrite backend

### Features

- 5 Theme variants
- Blog system with Markdown
- SSR & code splitting
- Authentication system

### Theme Options

1. Royal Purple (default)
2. Midnight
3. Sunset
4. Neon
5. Aurora
